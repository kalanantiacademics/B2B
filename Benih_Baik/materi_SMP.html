IDE 1

Layar Judul (Start Screen) üé¨

Buat tombol "Mulai" sebelum game dimulai.

Logika:

  • Saat Green Flag: Show tombol, Hide game.
  • Saat Tombol Klik: Broadcast [Mulai], lalu Hide tombol.
  • Saat terima [Mulai]: Show game.
IDE 2

Petunjuk Tersembunyi üîç

Klik objek (misal: Lukisan) untuk melihat petunjuk.

Logika:

  • Sprite Petunjuk: Hide di awal.
  • Sprite Lukisan: When clicked -> Broadcast [Show Hint].
  • Sprite Petunjuk: When receive [Show Hint] -> Show.
IDE 3

Ruangan Baru (Room 2) üö™

Pindah ke backdrop baru setelah pintu terbuka.

Logika:

  • Pintu: Broadcast [Masuk Room 2].
  • Stage: When receive [Masuk Room 2] -> Switch backdrop.
  • Sprite Room 1: Hide saat masuk Room 2.
IDE 4

Brankas Password üîí

Gunakan blok Sensing untuk meminta kode.

Ask [Masukkan Kode] and wait
If <(answer) = (1234)> then ...
IDE 5

Sistem Inventori üéí

Gunakan Variabel untuk mengingat barang.

  • Buat Variabel [Punya Kunci?]. Set ke 0 di awal.
  • Saat Kunci diklik: Set [Punya Kunci?] ke 1.
  • Pintu: If <(Punya Kunci?) = 1> then Open.

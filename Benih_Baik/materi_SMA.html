What is a Loop?

"Melakukan hal yang sama berulang-ulang selama syaratnya masih terpenuhi."

üîÑ

1. The 'FOR' Loop

Dipakai kalau kita TAU mau mengulang berapa kali.

for i in range(5):
    print("Halo!")
    print(f"Ini perulangan ke-{i}")

Bedah Kode:

  • range(5): Bikin urutan angka 0, 1, 2, 3, 4 (Total 5 angka).
  • i: Variabel penghitung (mulai dari 0).
  • Komputer akan menjalankan kode di dalam blok sebanyak 5 kali.

Mastering Range()

range(5)

0, 1, 2, 3, 4 (Stop sebelum 5)

range(1, 6)

1, 2, 3, 4, 5 (Mulai dari 1, Stop sebelum 6)

range(0, 10, 2)

0, 2, 4, 6, 8 (Lompat 2)

range(10, 0, -1)

10, 9, 8, ... 1 (Mundur)

2. The 'WHILE' Loop

Dipakai kalau kita GAK TAU mau ulang berapa kali, tapi tau syarat berhentinya.

baterai = 100

while baterai > 0:
    print("Main Game...")
    baterai = baterai - 10

print("HP Mati.")

‚ö†Ô∏è DANGER ZONE: Infinite Loop

Kalau syaratnya SELALU True, komputer bakal kerja selamanya sampai crash.

while True:
  print("Ini gak bakal berhenti!")

Solusi: Tekan tombol STOP (Kotak) di Colab.

What is a Function?

Input (Bahan)
‚û°Ô∏è
MESIN

Kayak blender. Masukin buah (Input) -> Blender (Function) -> Jus (Output).

Creating a Function

def sapa_teman():
    print("Halo bro!")
    print("Apa kabar?")

# Cara Panggil:
sapa_teman()
sapa_teman()

Anatomi Fungsi:

  • def: Kata kunci buat bikin fungsi.
  • nama_fungsi(): Nama bebas (asal gak spasi).
  • : dan Indentasi: Wajib!
  • Panggil Fungsi: Tulis namanya + (). Kalau gak dipanggil, dia diam aja.

Parameters (The Ingredients)

Biar fungsi kita pinter, kita kasih data pas dipanggil.

def sapa(nama):
    print(f"Halo {nama}, selamat datang!")

sapa("Budi")
sapa("Siti")

Output:

Halo Budi, selamat datang!

Halo Siti, selamat datang!

Return (The Result)

PRINT (Cuma Ngomong)

def tambah(a, b):
  print(a + b)

hasil = tambah(2, 3)
# hasil isinya KOSONG (None)

Cuma tampil di layar, gak bisa disimpan.

RETURN (Ngasih Hasil)

def tambah(a, b):
  return a + b

hasil = tambah(2, 3)
print(hasil * 2) # Bisa diolah lagi!

Nilainya dikembalikan ke variabel.

Key-Value Pair

KEY (Kunci)
"username"
‚û°Ô∏è
VALUE (Nilai)
"@antigravity"

"Kunci harus unik. Nilai boleh apa saja."

Creating a Dictionary

# Dictionary Kosong
data = {}

# Dictionary Isi
siswa = {
  "nama": "Budi",
  "umur": 17,
  "hobi": ["Game", "Tidur"]
}

Aturan Main:

  • Pakai kurung kurawal {}.
  • Format: "key": value.
  • Pemisah antar item pakai koma ,.
  • Value bisa berupa List juga lho!

Accessing Data

Kalau List pakai nomor urut, Dictionary pakai NAMA KUNCI.

siswa = {"nama": "Siti", "kelas": "10 IPA"}

print(siswa["nama"]) # Output: Siti
print(siswa["kelas"]) # Output: 10 IPA

‚ö†Ô∏è KeyError: 'alamat'

Artinya: Kamu cari kunci yang gak ada di kamus.

Modifying Dictionary

Tambah / Ubah

# Kalau key belum ada -> Nambah
siswa["alamat"] = "Jakarta"

# Kalau key sudah ada -> Update
siswa["nama"] = "Siti Aminah"

Hapus

# Hapus key tertentu
del siswa["kelas"]

# Kosongkan semua
siswa.clear()

Writing to File

# Cara Aman (with open)
# Otomatis close sendiri

with open("catatan.txt", "w") as file:
  file.write("Halo, ini file pertamaku!")
  file.write("\nBelajar Python seru.")

Bedah Kode:

  • "catatan.txt": Nama file.
  • "w": Mode WRITE (Tulis). Kalau file belum ada, dibuatkan. Kalau sudah ada, DITIMPA (Hapus lama).
  • \n: Enter (Garis baru).

Reading from File

Gimana cara baca file yang sudah kita buat?

with open("catatan.txt", "r") as file:
  isi = file.read()
  print(isi)

Output:

Halo, ini file pertamaku!

Belajar Python seru.

Appending (Nambah Data)

Mode "w" (Write)

Jahat. Dia hapus semua isi lama, ganti yang baru.

Mode "a" (Append)

Baik. Dia nambahin tulisan di paling bawah.

with open("catatan.txt", "a") as file:
  file.write("\nIni baris tambahan.")
